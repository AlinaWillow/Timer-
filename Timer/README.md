# :ghost: Timer beta
Set the time by using 3 botton.
1. Menu botton (Pin 6) 
2. increasing botton (Pin7)
3. decreasing botton (Pin8)
## Tools
- Arduino UNO R3
- LCD
- Breadboard
- Push/Pull botton
